# color selector demo

Shows usage of https://github.com/flaute/bootstrap-colorselector

Just
```sh
> git clone https://github.com/AlexKVal/color-selector-html
> cd color-selector-html
> python -m SimpleHTTPServer 8000
> open http://localhost:8000
```
